📱💻 Device Bridge · Phase 1

Not connected. Same bridge runs on every device you own — phone (Termux), Windows laptop, macOS, Linux. Install on each. Maya sees them all the same.

📦 Install on Samsung Fold 5 (Termux · 30s)

Open Termux on your phone and run:

bash <(curl -fsSL https://iamsuperio.cloud/phone-bridge/install.sh)

When it finishes it prints a token · then start the bridge with: maya-bridge

💻 Install on Windows laptop/desktop (30s)

Open PowerShell and run:

iwr https://iamsuperio.cloud/phone-bridge/install.ps1 -UseB | iex

Then launch from Start Menu → "Maya Bridge" (or auto-start on login if you said yes during install).

🐧 Install on macOS / Linux (30s)

Same one-liner as Termux:

bash <(curl -fsSL https://iamsuperio.cloud/phone-bridge/install.sh)

Launch with: maya-bridge (added to PATH).
