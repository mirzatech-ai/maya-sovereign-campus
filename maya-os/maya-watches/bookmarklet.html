🔍 Send to Maya

How to install (15 seconds)

  1. Make sure the bookmark bar is visible · in Chrome press Ctrl+Shift+B if it's hidden
  2. Drag the cyan pill above from this page onto your bookmark bar
  3. Done. You'll see a bookmark titled "🔍 Send to Maya" in your bar

How to use

  1. Go to ANY web page
  2. (Optional) highlight some text on the page
  3. Click the "🔍 Send to Maya" bookmark
  4. A small "✓ Maya got it" badge pops up top-right for ~2 seconds
  5. Maya OS Campus floor shows the packet · Maya's event log has the entry
What Maya gets: URL · page title · first H1 · 240-char text snippet from the page · any text you highlighted (up to 800 chars). Nothing else. Nothing automatic — you choose what to send by clicking. Less surveillance than the extension would have been; less friction too.

Phone Chrome (Samsung Fold 5)

Phones don't have a visible bookmark bar but you can still trigger this:

  1. Open this page on your phone: iamsuperio.cloud/maya-watches/bookmarklet.html
  2. Long-press the cyan pill → "Copy link address"
  3. In Chrome on phone: tap the URL bar → start typing "send to maya" → choose "Add new bookmark" → paste the link as URL → save
  4. On any page: tap URL bar → type "send to maya" → tap the bookmark suggestion that appears → done
Why not the extension? Google quietly deprecated the --load-extension flag in Chrome 137 as an anti-malware measure. Auto-installing unpacked extensions on standard Chrome no longer works — only Chrome Web Store extensions install programmatically. The bookmarklet is the friction-free alternative that survives this Chrome policy.
